# Instruções de Uso

## 1\. Iniciar o Gateway

```bash
cd ./GW-newprotocol-simple-main
node server.js
```

## 2\. Conectar o Rastreador

Deixe o rastreador se comunicar com o gateway ao menos 1 vez.

> **Recomendação:** Conecte somente 1 rastreador por vez pois os comandos serao enviados ao ultimo rastreador.
> 
## 3\. Iniciar o Servidor Web

```bash
cd ./mandarComando
node server.js
```

## 4\. Abrir a Interface

Abra o `index.html` no navegador.

> **Nota:** Mantenha o gateway e o servidor web rodando simultaneamente para enviar comandos via interface web.
